# DBD Trivia Game - Project Brainstorm

## What the Program Does

- Presents Dead by Daylight trivia questions to the player
- Multiple choice format (A, B, C, D)
- Tracks player progress through levels or score
- Provides lifelines to help answer questions
- Ends game when player answers incorrectly or completes all questions

## How It Will Work in the Terminal

- Text-based interface with clear prompts
- Player types the letter of their choice and presses Enter
- Shows feedback on correct/incorrect answers
- Displays current score/level after each question
- Menu for lifelines if the player chooses to use one

## Player Mode

- Single player

## Core Features (Must-Have)

1. Display questions and choices
2. Accept player input
3. Check answers and give feedback
4. Track score/level
5. Implement at least one lifeline

## Optional / Extra Features

- More lifelines (50/50, Skip Question, Ask the Chat)
- Sound or ASCII effects for correct/incorrect answers
- Store questions in a JSON file for easier expansion
